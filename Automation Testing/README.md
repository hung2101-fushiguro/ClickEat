# Automation Testing

Thư mục này chứa 2 phần theo yêu cầu nộp:
- `Exercise 5`: kiểm thử DemoQA Practice Form (POM)
- `Exercise 6`: kiểm thử hệ thống ClickEat của nhóm (POM + System Test)

## Cách tổ chức
- Mỗi Exercise là một Maven test project độc lập
- Chạy test trong từng thư mục exercise bằng `mvn test`
